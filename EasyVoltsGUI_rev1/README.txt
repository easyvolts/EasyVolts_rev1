EasyVoltsGUI is a convenient UI for EasyVolts device. It provides convenient setup of voltage and current limit and easily readable measured load voltage and current.
This is a Python3.5.2 project. UI is build with help of wxFormBuilder 3.6.0 (*.fbp file). Generated by the builder software in in EasyVoltsGUI.py file. Business logic is locted in EasyVoltsGUI.pyw file.
